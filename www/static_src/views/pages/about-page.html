About the Project

Open Data Discovery is a project to track and present the up-to-date status of the world of open data. It collects information from {{ about.portalCount | number }} open data portals: dataset number, tags, categories, publishers, and more in a weekly basis to show how they are acting. These data portals are established by nations, provinces, cities, and institutes worldwide, which make up the main force of the global activity of data openning.
